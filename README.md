# NOVA KICKS 👟

**NOVA KICKS** - демонстрационный интернет-магазин кроссовок с современным темным интерфейсом, адаптивной версткой и полноценным пользовательским сценарием от просмотра каталога до оформления демо-заказа.

Проект создан как пет-проект для портфолио и показывает реализацию аккуратного frontend-приложения на React + TypeScript: с роутингом, централизованным состоянием корзины, фильтрацией товаров, переключением валют и интерактивными UI-сценариями.

Посмотреть его сейчас можно по данной ссылке: https://shoes-store-five-mocha.vercel.app/

## ✨ Обзор

NOVA KICKS имитирует небольшой sneaker-store: пользователь может просматривать каталог, фильтровать товары по категориям, открывать подробную карточку кроссовок, добавлять позиции в корзину, менять количество товаров и переходить к оформлению заказа.

Интерфейс выдержан в темной zinc/emerald-палитре: компактные карточки, акцентные CTA-кнопки, адаптивные сетки и визуальный фокус на товаре.

## 🚀 Функциональность

- 🛍️ Каталог кроссовок с фильтрацией по категориям.
- 🧾 Карточки товаров с фото, ценой, старой ценой и кнопкой добавления в корзину.
- 🔍 Модальное окно товара с описанием, материалами, особенностями, посадкой и zoom-эффектом изображения.
- 🛒 Корзина с изменением количества, удалением товаров и пересчетом итоговой суммы.
- 💱 Переключение валют: `UAH`, `USD`, `EUR`.
- ⏳ Акционный hero-блок с таймером до конца недели.
- 📦 Страница оформления заказа с формой доставки и выбором способа получения.
- ✅ Демо-состояние успешного оформления заказа.
- 🏪 Страница About с презентацией концепции магазина.
- 📱 Адаптивная верстка для desktop, планшетов и мобильных экранов.

## 🗂️ Структура

```text
src/
  components/   переиспользуемые UI-компоненты
  data/         статические товары, категории и валюты
  hooks/        пользовательские React-хуки
  logic/        форматтеры и чистая бизнес-логика
  pages/        страницы приложения
  styles/       глобальные стили
  types/        общие TypeScript-типы
```

Ключевое состояние приложения находится в `src/App.tsx`: корзина, выбранная валюта и активная категория. Такой подход сохраняет простую архитектуру и делает поток данных понятным для SPA этого масштаба.

## 🧭 Маршруты

- `/` - главная страница с hero-блоком, каталогом и корзиной.
- `/about` - страница о магазине.
- `/checkout` - оформление демо-заказа.

## ⚙️ Запуск проекта локально

Установка зависимостей:

```bash
npm install
```

Запуск dev-сервера:

```bash
npm run dev
```

Production-сборка:

```bash
npm run build
```

Preview production-сборки:

```bash
npm run preview
```
